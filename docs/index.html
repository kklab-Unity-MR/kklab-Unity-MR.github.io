スキル

C

C言語の基本文法．ソートなどのアルゴリズム実装．動的なメモリ管理．CUIベースのゲーム開発．

C#

Unityでの開発のためのメイン言語．Unity内のプロジェクトでオブジェクトなどを制御するプログラムを書くことができる．

Python

Pythonの基本文法．機械学習(ニューラルネットワークや自然言語処理)．クラスタリングなどのデータ分析．Djangoフレームワークを用いたWeb開発

Java

javaの基本文法．簡易的なゲーム開発．Spring Bootを用いたLineBot開発

GAS

Googleのサービス(GoogleDriveやスプレッドシートなど)を連携したWeb開発．簡易的なLineBot作成．

DataBase

サービス連携と連携するために使用．MysqlやPostgreSQLやH2 DataBaseを利用した．

Unity

Unityの基本操作や知識持っている．主にMR(HoloLens2)のアプリ開発のために使用中．

Git

基本的なGitコマンドを使用できる．Gitを用いた個人でのプロジェクト管理やチーム開発経験有あり．

AWS

AWSを用いたクラウド開発やサービス連携．具体的にはEC2，RDS，Amazon Rekognitionなどのサービスを利用したことがある．

Linux

サブのオペレーティングシステムとして使用中．主に研究室のサーバやメーリングリスト(fml)の管理に使用．
